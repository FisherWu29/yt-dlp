# 快速启动步骤

## 启动服务

1. 进入 app 目录：
   ```bash
   cd /Users/fisher/fisher/MyProjects/VideoSafe/yt-dlp/app
   ```

2. 激活虚拟环境：
   ```bash
   source venv/bin/activate
   ```

3. 启动服务：
   ```bash
   python app.pyß
   ```·

## 访问地址

- 服务地址：http://localhost:8000
- API 文档：http://localhost:8000/docs

## 停止服务

在运行服务的终端中，按下 `Ctrl+C` 即可停止服务。·
